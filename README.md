# Apple Music (apple-music)

Apple Music is Apple's streaming music service. The Apple Music API and MusicKit framework give developers access to the Apple Music catalog, user library, charts, recommendations, search, and playback control.

## APIs

### Apple Music API
REST API for catalog (songs, albums, artists, music videos, playlists, stations), search, charts, recommendations, and the authenticated user's library and ratings.
- **Base URL:** `https://api.music.apple.com/v1`
- **Auth:** Apple Developer Program JWT token (signed with team key); Music User Token for personal library
- **Docs:** https://developer.apple.com/documentation/applemusicapi
